# group-circle

아래 문서는 mac os 기준으로 작성됨

# 개발환경 버전

node : v8.11.3
npm : 5.6.0
express : 4.16.0

# Getting start

## install node

1. Enter https://nodejs.org/ko/ and download node 8.11.3 LTS
2. install webpack

- npm install webpack webpack-cli --save-dev
- npm install babel-core babel-loader babel-preset-es2015 --save-dev

## build

~~~shell
cd group-circle
npm install
npm start
~~~
